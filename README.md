# RGB Color Picker 
A simple RGB color picker built with vanilla JavaScript, created to demonstrate core front-end concepts

Live demo [here](https://isntnot.github.io/color_picking/)

---

##  Features

- Real-time color updates using RGB sliders
- Synchronized range sliders and number inputs
- Color state persistence using `localStorage`
- Clean, minimal UI
- Fully client-side (no dependencies)

---

## Concepts Demonstrated

This project showcases practical JavaScript fundamentals:

- DOM manipulation
- Event handling (`input` events)
- State synchronization between UI elements
- Browser storage (`localStorage`)

---

## Built With

- HTML5
- CSS3
- Vanilla JavaScript (ES6)

No frameworks or libraries were used.
